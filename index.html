Get to know us

Hey! I'm Georgiana Nistor , a passionate FrontEnd Developer/Trainer. Recently this month I finished teaching a 6-month web development course were I've touched several FrontEnd topics: HTML, JavaScript, CSS, React but also Git and GitHub. As an exercise for Git, I challenged my students to fork this GitHub repository and most of them filled their contact details via a Pull Request.

Have a look at their LinkedIn, GitHub and CodePen, profiles, and if you like what you see, don't hesitate to contact them for a job!
